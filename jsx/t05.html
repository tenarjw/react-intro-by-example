Bardziej złożone elementy zapisujemy w postaci obiektów .
Zwróć uwagę, że props stają się własnością obiektu (użycie this.) a renderowanie metodą (render()).
https://toddmotto.com/react-create-class-versus-component/ 
class Hello extends React.Component {
 render() { 
  let data=(new Date()).toDateString();
  return(
   <div>
    <h1>Hello {this,props.toWho}!!!!</h1>
    <p>Data: {data}</p>
   </div>); 
 }
}

let element=<Hello toWho='World' />;
let div=document.getElementById('root');
ReactDOM.render(element,div);
